## 卒研で学んだこと
1. やる気がなくても定期的にレポジトリを掃除しましょう。or データの作成間違って詰む
2. 考えはできるだけちゃんと「図」にまとめて持っていくと、言語の壁を越えて理解されやすい
3. コードの実行を面倒くさがってデータを別保存すると、根本的なミスをしたときに詰む。
4. 周りの人間の言葉をうのみにしていると何も進まない。
